## Problem Statement

Production deployment had multiple issues:
1. Neon PostgreSQL cold-start causing 500 errors on page load
2. Resend emails rejected (403) due to `FROM_EMAIL` domain mismatch (.com vs .ca)
3. Admin dashboard crashing from Canada (4 parallel DB queries on cold-start)
4. Admin notification email formatting (gapping/spacing issues)
5. No rate limiting on contact forms (quote/career submissions)
6. No user authentication for form submissions

## Changes Made

### 1. Neon Connection & Cold-Start Fix

- **`lib/prisma.ts`** — Added `withDbRetry()` wrapper (retries query once after 500ms)
- **`prisma/schema.prisma`** — Added `relationMode = "prisma"` for PgBouncer compatibility
- **`.env`** — Added `?pgbouncer=true` to `DATABASE_URL`
- All 4 GET API routes (projects, services, collaborations, testimonials) wrapped with `withDbRetry`
- Front-end components (`app/page.tsx`, admin pages) added `Array.isArray(data)` guards

### 2. Resend Email Fix

- **`.env`** — Changed `FROM_EMAIL` from `noreply@newcanadiandrywall.com` → `noreply@newcanadiandrywall.ca`
- Verified via Resend logs: all previous sends returned 403 (domain mismatch), fixed sends return 200

### 3. Admin Email Formatting

- **`lib/email.ts`** — Redesigned admin notification HTML templates:
  - Added `cellpadding="0" cellspacing="0"` to fix gapping in email clients
  - Increased row padding from 8px to 10px with border-bottom separators
  - Message field in separate card with `#f9f8f6` background
  - Added `Years` suffix to experience field (capitalized)

### 4. Admin Notification Logic

- **`app/api/contact/quote/route.ts`** — Admin notification gated on `submission.status === 'new'`
- **`app/api/contact/career/route.ts`** — Same gating logic

### 5. Admin Dashboard Cold-Start Fix

- **`app/admin/(dashboard)/page.tsx`** — Wrapped all 4 `prisma.count()` queries with `withDbRetry`

### 6. Rate Limiting

- **`lib/rate-limit.ts`** — Rewrote from cookie-based to server-side IP-based rate limiting with in-memory Map:
  - Login: 5 attempts / 15 min
  - Quote form: 3 submissions / 1 hour
  - Career form: 3 submissions / 1 hour
- **`app/api/contact/quote/route.ts`** — Added `checkFormRateLimit()`
- **`app/api/contact/career/route.ts`** — Added `checkFormRateLimit()`
- **`app/api/auth/login/route.ts`** — Updated to use new `checkLoginRateLimit()`

### 7. User Authentication (NextAuth v5)

New files created:
- **`lib/auth-config.ts`** — NextAuth v5 config with Google OAuth + Resend email magic link
- **`app/api/auth/[...nextauth]/route.ts`** — Auth API handler
- **`app/auth/signin/page.tsx`** — Sign-in page with "Continue with Google" button + email magic link
- **`component/SessionProvider.tsx`** — Client-side session provider wrapper
- **`prisma/schema.prisma`** — New models: `User`, `Account`, `Session`, `VerificationToken`
  - Added `userId` field to `QuoteSubmission` and `CareerSubmission`
