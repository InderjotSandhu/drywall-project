## Changes Made

### 1. Admin Logout Redirect Fix

- **`app/api/auth/logout/route.ts`** — Changed redirect from `process.env.NEXT_PUBLIC_SITE_URL || 'http://localhost:3000'` to `request.url` so it always redirects to the same origin
- **`component/AdminNav.tsx`** — Refactored from native `<form action="/api/auth/logout" method="POST">` to `fetch` POST + `window.location.href` redirect to prevent native form navigation issues

### 2. Login Page Error Handling

- **`app/admin/login/page.tsx`** — Added explicit `401` check in useEffect fetch to prevent console error spam; added `.catch(() => {})` for network errors; added `action="javascript:void(0)"` to form to prevent native POST to page route

### 3. Mobile Menu Positioning

- **`component/Navbar/Navbar.module.css`** — Changed `.mobileMenu` from `position: absolute` with `top: 100%` to `position: fixed` with explicit `top: 68px` (desktop) / `top: 56px` (mobile) for consistent positioning across devices with browser chrome

### 4. Services Dropdown on Mobile

- **`component/Services/Services.tsx`** — Removed hardcoded `COLS = 3` row-grouping logic. Changed to flat grid rendering where the detail panel is placed directly after the clicked card in the DOM. CSS grid auto-places correctly regardless of screen width
- **`component/Services/Services.module.css`** — Added `touch-action: manipulation` for better touch responsiveness; removed `overflow: hidden` from grid container which was clipping the expanded panel on mobile

### 5. Resend Email Fix (Unawaited Promises)

- **`app/api/contact/quote/route.ts`** — Wrapped `Promise.allSettled([...])` in `await` so emails complete before serverless function terminates
- **`app/api/contact/career/route.ts`** — Same fix applied

### 6. Admin Email Fallback Fix

- **`lib/email.ts`** — Changed fallback emails from `.com` to `.ca`:
  - `noreply@newcanadiandrywall.com` → `noreply@newcanadiandrywall.ca`
  - `info@newcanadiandrywall.com` → `info@newcanadiandrywall.ca`

### 7. Testimonials Marquee Pause-on-Hover

- **`component/Testimonials/Testimonials.tsx`** — Added `paused` state to `MarqueeRow` component with `onMouseEnter`/`onMouseLeave` handlers to toggle `.marqueePaused` class
- CSS class `.marqueePaused` with `animation-play-state: paused` was already defined

### 8. Performance Audit
